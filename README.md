# Odisha Yatri: A Comprehensive Data Analytics Project on Ride-Booking Platform

## 📌 Overview
Odisha Yatri is an end-to-end data analytics project built on ride-booking platform 
data, covering the complete analytics pipeline — from raw data cleaning to 
interactive business dashboards. The project demonstrates practical application 
of Excel, SQL, and Power BI to derive actionable insights from real-world 
transactional data.

## 🛠️ Tools & Technologies
- **Microsoft Excel** – Data cleaning, preprocessing, and exploratory analysis
- **PostgreSQL** – Data querying, transformation, and analysis using SQL
- **Power BI** – Interactive dashboards and data visualization
- **DAX** – Custom measures and calculated KPIs for business insights

## 🔄 Project Workflow
1. **Data Cleaning (Excel)** – Handled missing values, inconsistencies, and 
   formatting issues in raw ride-booking data
2. **Data Querying (PostgreSQL/SQL)** – Wrote structured queries to extract, 
   join, and analyze ride, driver, and customer data
3. **Dashboard Development (Power BI)** – Built interactive dashboards with 
   DAX measures to visualize key metrics like ride volume, revenue trends, 
   cancellation rates, and customer behavior

## 📊 Key Insights
- Ride booking trends across time periods
- Revenue and cancellation analysis
- Customer and driver performance metrics

## 🎓 Academic Context
This project was developed as a final-year MCA capstone project under the 
Data Analytics training program, as part of coursework at the Institute of 
Professional Studies and Research (IPSAR), Cuttack.
